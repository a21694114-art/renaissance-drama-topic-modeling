# Interactive Visualization Results

Large Plotly HTML visualizations are not fully included in this repository because of file size limitations.

A lightweight sample visualization is provided in this directory for demonstration purposes.

## Full Interactive Visualizations

The complete interactive topic visualizations are hosted externally at:

https://renaissance-bertopic.netlify.app/

These visualizations include interactive exploration of:

- topic distributions
- author highlighting
- genre highlighting
- title highlighting
- temporal patterns
- topic-level semantic structures

The visualizations were generated using Plotly based on the chunk-level master table and the reduced UMAP coordinates produced during the clustering stage.
