# Shakespeare vs Non-Shakespeare Comparative Visualization

This stage generates comparative genre-level topic visualizations for Shakespearean and non-Shakespearean drama.

Using the normalized genre-topic percentage tables constructed during the comparative analysis stage, the script produces parallel visualizations for:
- Comedy
- Tragedy
- History

For each corpus group, the top 20 topics are visualized using:
- stacked bar charts
- horizontal topic-distribution charts

Topic summaries generated during the topic-labeling stage are integrated directly into the figures to improve interpretability.

The visualizations form the basis for the dissertation’s comparative analysis of thematic distributions across Shakespearean and non-Shakespearean dramatic corpora.
