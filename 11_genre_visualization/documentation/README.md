# Genre-Level Visualization

This stage generates genre-level topic distribution visualizations based on the normalized topic-percentage table.

The visualizations focus on the three major dramatic genres:
- Comedy
- Tragedy
- History

For each genre, the top 20 most prominent topics are identified and visualized using stacked bar charts and horizontal bar charts.

Topic summaries generated during the topic-labeling stage are integrated directly into the figures to improve interpretability.
